# APEX-Macro
# 扣群1084400157
APEX 手柄/转换器压枪宏  无需导入无需steam映射 打开即用
1.先把自己原来的罗技驱动卸载删掉，然后装里面的罗技驱动 ”罗技驱动（拖到桌面安装.exe）“

2.接着打开 TNT.exe 就可以直接使用了  

3.若想自定义映射可自行设置(不知道自己手柄键位对应哪个粉色方块 按自己手柄按键会有闪烁提示)

5.游戏内的灵敏度设置可以看”游戏设置.png“。 

”不需要导入数据，不需要自己映射“

程序不涉及任何注入，纯原生映射+调用罗技驱动
程序仅供学习交流
